Dynamic Multiplication Table

Enter values in the forms to create a multiplication table. The bounds can be switched if necessary. Each form must have a valid integer. Click save table to save the current multiplication table. Delete all tabs will delete all the saved tables. There can be a maximum of 10 saved tables.
